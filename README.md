# README

## 介護記録共有アプリ Care_diary

### 概要
このアプリは大切なペットたちの毎日の介護記録を保存できます。
家族等、グループ間での記録の共有ができます。

### バージョン
- Ruby 2.6.5
- Ruby on Rails 5.2.4
- PostgreSQL 13.0

### 課題要件
#### カリキュラム内の技術2つ
- devise
- コメント機能
#### カリキュラム外の技術１つ
- gem ransack
